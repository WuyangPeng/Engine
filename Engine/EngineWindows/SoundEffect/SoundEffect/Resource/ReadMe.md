音效库模块

音效库（SoundEffect）封装了引擎的音效功能。主要对音效的调用接口进行封装。
音效库位于引擎的第三层，只允许调用引擎中的CoreTools库、System库，并使用boost和其他音效相关的工具库。 

----------------------------
1.	Helper（帮助）

音效库帮助模块。

（1）用户宏：用于是否使用静态库、是否使用OpenAL等用户设定。
（2）音效库宏：音效库模块所使用的宏，按需要添加。

----------------------------
2.	Configuration （配置）

封装了音效库的配置。

（1）配置策略：对音效库的相关策略进行封装，按需要后续添加。 
（2）配置解析：对音效库配置进行解析，返回配置的策略。

----------------------------
3.	Interface（接口）

封装了音效库的统一接口。 处理模式使用策略类，分别代表使用哪种音效库。 
 
（1）调用：实现各音效库的统一调用接口。

----------------------------
4.	CsynUtil（CsynUtil） 

封装了调用CsynUtil代码的实现。

（1）调用：实现CsynUtil的调用接口。

----------------------------
5.	DirectSound（DirectSound）

封装了调用DirectSound代码的实现。

（1）调用：实现DirectSound的调用接口。

（1）调用：实现ogg的调用接口。
