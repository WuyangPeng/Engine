人工智能库模块

人工智能库库（ArtificialIntellegence）封装了引擎的人工智能库层调用。以实现人工智能库层的封装。
人工智能库库位于引擎的第五层，只允许调用引擎中的CoreTools、Mathematics、Rendering、Physics、Imagics库，并使用boost和其他人工智能库相关的工具库。
