数学库模块

数学库（Mathematics）封装了引擎的数学功能。以实现线性代数相关的计算。
数学库位于引擎的第三层，只允许调用引擎中的System、CoreTools库，并使用boost工具库。 

----------------------------
1.	Helper（帮助）

数学库模块系统宏。

（1）用户宏：用于是否使用静态库、是否使用导出模板、特定断言等用户设定。

----------------------------
2.	Base（基础）

封装了数学库所需要的一些底层实现。如截断输入参数到某一范围，避免当输入略有误差而产生NaN的问题。通过使用类的静态成员函数实现。

（1）位操作：包含截断输入参数到某一范围，幂的快速计算。
（2）数学函数：包含三角函数快速计算。
（3）数学工具：包含数学工具的基础实现。
（4）浮点组：封装了浮点数组的使用。

9.	Query（查询）

关于一个点和各种几何对象之间的关系查询。包括2维对象同线、三角形、三角形外切圆之间的关系，3维对象同平面、四面体、四面体外切球之间的关系。

（1）  2D查询：获取2维对象同线、三角形、三角形外切圆之间的关系。
（2）  3D查询：获取3维对象同平面、四面体、四面体外切球之间的关系。
