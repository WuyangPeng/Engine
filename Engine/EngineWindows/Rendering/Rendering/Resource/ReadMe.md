----------------------------
渲染库模块

渲染库（Rendering）封装了引擎的渲染库调用。主要对渲染的调用接口进行封装。
渲染库位于引擎的第四层，只允许调用引擎中的System、CoreTools、Mathematics库，并使用boost和其他渲染相关的工具库。  

----------------------------
1.	Helper（帮助）

渲染库帮助模块。
 
（1）用户宏：用于判断是否使用静态库及其他用户设定。 

----------------------------
2.	Base（基础）

渲染库的基础模块。包括图形对象基类。
 
（1）图形对象基类：渲染库对象基类。 

----------------------------
3.	DataTypes（数据类型）

渲染库的数据类型。包括实用工具、颜色管理、半浮点数、转换和范围类。
 
（1）实用工具：通过x轴或y轴反射颜色数组。
（2）颜色管理：指定的格式和32位RGB颜色之间的转换，并实现一个基础的颜色类。
（3）半浮点数：封装一个半浮点数。
（4）纹理颜色：通过纹理格式自动定义的颜色类。
（5）边界：一个球体边界类，指定为一个中心和半径的球体。
（6）转换：封装转换类。

----------------------------
4.	Resources（资源）

渲染库的资源类。包括顶点缓冲区、索引缓冲区、纹理对象、渲染目标以及顶点格式。
 
（1）顶点缓冲区：实现顶点缓冲区的读取和写入。
（2）索引缓冲区：实现索引缓冲区的读取和写入。
（3）纹理对象：实现1D、2D、3D和Cube纹理对象。 
（4）渲染目标：实现渲染目标的数据封装。
（5）顶点格式：实现顶点格式的数据封装。

----------------------------
5.	SceneGraph（场景图）

包含抽象渲染API、相机对象、光照、纹理和图像、帧缓冲区和全局渲染状态，包括混合操作、面剔除操作、材质、多边形偏移、目标缓冲区状态、线框模式以及深度缓冲区机制。
支持场景图管理行为，包括树形结构、内部节点、叶节点、点-粒子图元、直线图元、三角形图元、包围体、转换操作、层次结构剔除机制、顶点缓冲区以及索引缓冲区。
 
（1）  相机对象：实现相机对象。
（2）  节点对象：实现节点对象。
（3）  光照对象：实现光照对象。
（4）  材质对象：实现材质对象。
（5）  点-粒子图元：实现点-粒子图元。
（6）  直线图元：实现直线图元。
（7）  三角形图元：实现三角形图元。
（8）  转换操作：实现转换操作。
（9）  层次结构剔除机制：实现层次结构剔除机制。
（10） 屏幕目标：实现屏幕目标。
（11） 标准网格：实现标准网格。
