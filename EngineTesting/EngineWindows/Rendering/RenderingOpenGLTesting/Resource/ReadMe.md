渲染库OpenGL单元测试

1.	RendererEngineSuite

对渲染引擎模块的单元测试，包括： 
（1） 渲染基础：EnvironmentParameterTesting、
				RenderingEnvironmentTesting。
（2） 渲染参数。
（3） 顶点格式对象。
（4） 索引缓冲区。
（5） 顶点缓冲区。
（6） 纹理对象。
（7） 缓冲管理。
（8） 渲染目标。

----------------------------
2.	OpenGLRendererSuite

对OpenGL渲染器模块的单元测试，包括： 
（1）OpenGL映射。
（2）OpenGL纹理。
（3）OpenGL顶点格式。
（4）OpenGL渲染对象。
（5）OpenGL位图。
（6）OpenGL缓冲区。
