资源管理库模块

资源管理库（ResourceManager）封装了引擎的资源管理。包含三维模型资源、纹理资源、材质资源、字体资源、骨骼资源、碰撞资源、物理参数和游戏世界和地图。
脚本库位于引擎的第四层，只允许调用引擎中的CoreTools库、Mathematics库，并使用boost和其他资源管理相关的工具库

----------------------------
1.	Helper（帮助）

（2）资源管理库宏：资源管理库模块所使用的宏，按需要添加。
