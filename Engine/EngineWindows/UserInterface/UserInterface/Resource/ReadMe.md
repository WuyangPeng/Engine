1.	Helper（帮助）

待实现功能：
（1）用户宏：用于是否使用静态库及其他用户设定。
（2）用户界面库宏：用户界面库模块所使用的宏，按需要添加。

----------------------------
2.	Configuration （配置）

----------------------------
