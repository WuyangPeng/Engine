----------------------------
用户界面库模块

----------------------------
1.	Helper（帮助）

用户界面库帮助模块。

（1）用户宏：用于是否使用静态库及其他用户设定。
（2）用户界面库宏：用户界面库模块所使用的宏，按需要添加。

----------------------------
2.	Configuration （配置）

封装了用户界面库的配置。

（1）配置策略：对用户界面库的相关策略进行封装。 
（2）配置解析：对用户界面库配置进行解析，返回配置的策略。

----------------------------
3.	Interface（接口）

封装了用户界面库的统一接口。 处理模式使用策略类，分别代表使用哪种用户界面库。

----------------------------
4.	wxWidgets（wxWidgets）

封装wxWidgets。

----------------------------
5.	FLTK（FLTK）

封装FLTK。

----------------------------
6.	OpenGL（OpenGL）

封装OpenGL。

----------------------------
7.	DirectX（DirectX）

封装DirectX。
