特别版
★★
作者：Bjarne Stroustrup

知识点 
1.	C++ 基础知识。
2.	C++ 基础知识练习。

代码 
1.  书籍代码。
2.  练习代码。

要点 
1.	避免非平凡的指针算术。
2.	如果你必须使用宏，请使用带有许多大写字母的丑陋的名字。
3.	在构造全局的和名字空间的对象时，应避免顺序依赖性。
4.	将只有一个“大小参数”的构造函数做成explicit。

第4版第1-3部分
★★★★
作者：Bjarne Stroustrup

知识点 
1.	C++ 基础知识。
2.	C++11 基础知识。
3.	一个矩阵设计。

要点 
1.	将指针看作Java的引用在C++中的等价物。
2.	以值的方式返回容器（依赖于移动操作以提高效率）。
3.	数据结构应力求小巧。
4.	注释应该简短直接。
5.	库不应直接输出面向最终用户的错误诊断信息，正确的做法是抛出一个异常然后由调用者决定该怎么做。
