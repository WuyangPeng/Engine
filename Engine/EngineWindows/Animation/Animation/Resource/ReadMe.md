动画库模块

动画库（Animation）封装了引擎的动画库层调用。以实现动画库层的封装。
动画库位于引擎的第五层，只允许调用引擎中的CoreTools、Mathematics、Rendering库，并使用boost和其他动画库相关的工具库。

----------------------------
1.	Helper（帮助）

动画库帮助模块。 
（1）用户宏：用于判断是否使用静态库及其他用户设定。
