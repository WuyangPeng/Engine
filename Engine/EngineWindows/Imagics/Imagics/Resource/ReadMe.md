图像分析库模块

图像分析库（Imagics）封装了引擎的图像分析模块，包含2D和3D图像分析处理的基本例程，如从2D图像析取各级曲线，或从3D图像中析取各级表面。
图像分析库位于引擎的第四层，只允许调用引擎中的System、CoreTools、Mathematics库，并使用boost和其他图像分析相关的工具库。

----------------------------
1.	Helper（帮助）

图像分析库模块系统宏。
 
（1）用户宏：用于是否使用静态库及其他用户设定。
（2）图像分析库宏：图像分析库模块所使用的宏，按需要添加。
（3）元素宏：自然类型的包装。
（4）图像转换器宏：图像转换器函数的封装。

----------------------------
